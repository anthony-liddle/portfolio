# Contributing

This is a personal portfolio site. I'm not accepting feature PRs, but **issues are welcome** for:

- Typos or factual errors in case study content
- Accessibility bugs
- Broken links
- Rendering issues on specific browsers or devices

## Development setup

```bash
pnpm install
pnpm dev
```

Requires Node.js 20+ and pnpm.

## Commit conventions

This repo uses [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add new case study
fix: correct broken link in Pocket case study
style: adjust heading size on mobile
```
